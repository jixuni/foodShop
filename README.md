# foodShop

Recipe app build with es6 conventions and MVC architecture. 

## Getting Started

````
clone the the repo
npm run dev (Will start the development server)
````

### Prerequisites

Chrome or Mozilla, have not tested any other browsers.

## Deployment

````
npm run build (will bundle the javascript) 
````

## Built With

* JavaScript
* HTML/CSS 
* Babel
* Webpack
* Axios
